Deforestation Early Warning System

Project Overview

This project created an automated system for detecting deforestation events in near-real-time using satellite imagery. The system processes daily Sentinel-2 imagery and detects changes in forest cover using a combination of spectral indices and machine learning algorithms.

When potential deforestation is detected, the system generates alerts with geographic coordinates, estimated area, and confidence level. These alerts are sent to forest protection authorities via a web platform, email, and mobile app notifications.

Technical Approach

  • Automated data acquisition from Sentinel-2 satellite via Google Earth Engine API
  • Cloud detection and removal using the Sentinel-2 quality bands
  • Time series analysis to distinguish between seasonal changes and deforestation
  • Random Forest classifier for identifying deforestation events
  • Alert system with web platform and mobile app integration

Results & Impact

The system has been operational since January 2024 and has successfully detected over 85% of deforestation events with a false positive rate of less than 10%. It covers an area of approximately 1.2 million hectares of protected forest and has contributed to a 30% reduction in response time to deforestation events.
